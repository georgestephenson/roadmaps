# Claude Code Roadmap

Notes from studying the [Claude Code Roadmap](https://roadmap.sh/claude-code).

Two days after I starting this roadmap, [the entire Claude Code source code leaked online](https://news.ycombinator.com/item?id=47584540).

As such, the resources for this roadmap are suddenly missing the best resource of all: the source code itself. It already feels outdated!

[Claude Code Unpacked](https://ccunpacked.dev/) is a great quick overview of findings from the leak. For example, it explains the "Agentic Loop" in Claude Code but links back to the specific code files where these Thought-Action-Observation steps happen.
